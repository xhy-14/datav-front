<br>

安装

npm run dev

<h1>目录结构</h1>

```
datav
├─public  项目公共模块
│
├─src 源路径
│  ├─api 请求接口包
│  ├─assets 静态资源包
│  ├─components 自定义组件包
│  ├─router 路由包
│  ├─sotre  仓库
│  ├─utils 工具包
│  └─views 视图包

```
